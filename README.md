# BoostedAudio

BoostedAudio is a web audio client system for minecraft server
Including:
- Music/Sound effect system
- Realtime proximity chat
- Client customisation
- Speaker system
- Spatialized audio


Creating/convert ssl:
https://www.baeldung.com/convert-pem-to-jks
